# BriaNova Software Solutions

Sitio web corporativo de BriaNova Software Solutions con panel de administración.

## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/BriaNova-Software-Solutions/brianova.git
cd brianova
```

2. Instalar dependencias:
```bash
npm install
```

3. Iniciar el servidor:
```bash
npm start
```

4. Abrir en el navegador:
```
http://localhost:3000
```

## Acceso al Panel de Administración

El panel de administración está disponible en:
```
http://localhost:3000/admin/login
```

## Estructura del Proyecto

```
brianova/
├── admin/              # Archivos del panel de administración
├── data/              # Datos y configuraciones
├── public/            # Archivos estáticos
├── server.js          # Servidor Express
├── package.json       # Dependencias y scripts
└── README.md          # Este archivo
```

## Tecnologías Utilizadas

- HTML5
